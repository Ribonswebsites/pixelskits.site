Transparency

Smart Processing Tools

Deterministic algorithm
Auto-Enhance · Sharpen · Denoise · Cinematic Grade
ONNX ML model
Remove Background — genuine neural network

Auto-Enhance, Sharpen Filter, Denoise Filter, and Cinematic Grade use deterministic client-side algorithms — histogram normalisation, unsharp mask convolution, bilateral filtering, and tonal curve simulation. They are not machine learning models.

Remove Background (in-browser mode) is the sole tool using a genuine ML model — a pre-trained ONNX segmentation model running locally via WebAssembly. No image data is sent to any server in this mode. All processing occurs entirely in your browser. Results may vary based on device performance, memory, and browser version.
